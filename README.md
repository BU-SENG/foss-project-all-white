## CAMPUS MARKETPLACE
An online campus marketplace where students can buy and sell goods to their fellow students.

## FEATURES 
-   Student account verification
-   Buy and sell items within the campus
-   Item search and categorization
-   Secure messaging

## PREREQUISITES
You need to have all these installed for everything to work fine:
-   **Node.js** - JavaScript run-time environment
-   **Git** - Version Control System
-   **VSCode** - Integrated Development Environment

## HOW TO USE THIS REPO 
1. Clone the project. Open your terminal and use the following command:
    ```bash
    git clone https://github.com/BU-SENG/foss-project-all-white.git
    ```
2. Open the "foss-project-all-white" folder in VS Code (or your preferred code
   editor)
3. Open a new terminal in the same directory as the above project. Then run the
   command:
    `npm install`
4. Run the application server:
    `npm run dev`
5. View the page in the browser with the link provided in the terminal.
