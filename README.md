This project is an online campus market place, where students can buy and sell goods to their fellow students.
The goal of this project are:

Enables low-cost buying/selling	Reduces financial stress,
Verified student-only trading	Builds trust & accountability,
Encourages open-source participation	Builds digital skills.

Features of this project

Student account verification
Buy and sell items within the campus
Item search and categorization
User profiles
Secure messaging or meet-up coordination
